# BOLLARD Web Course (Editable HTML)
Цвет бренда: RGB(0,128,128) / #008080

Запуск (рекомендуется через локальный сервер, чтобы работал fetch):
- Python: `python -m http.server 8000`
- Открыть: http://localhost:8000/

Редактирование:
- data/course.json — модули/страницы/тесты (RU/EN)
- data/i18n.json — тексты интерфейса (RU/EN)
- css/styles.css — стиль интерфейса
- certificate.html + css/certificate.css — сертификат
- assets/logo.png — логотип

Прогресс:
- хранится в localStorage (в этом браузере).
